# ENT Research Data Collection — PWA

Installable on **Android**, **iOS**, and **Windows** directly from the browser. No app store required.

## Live URL
```
https://likhithdhruva.github.io/ENT-RESEARCH-0/RESEARCH_APP/
```

---

## Install Instructions

### Android (Chrome)
1. Open the URL above in Chrome
2. Tap the **⋮ menu → "Add to Home screen"**
3. Tap **Install** — app appears on your home screen

### iOS (Safari)
1. Open the URL in **Safari** (must be Safari, not Chrome)
2. Tap the **Share button (□↑) → "Add to Home Screen"**
3. Tap **Add** — app icon appears on home screen

### Windows (Chrome / Edge)
1. Open the URL in Chrome or Edge
2. Click the **install icon (⊕)** in the address bar, or go to **⋮ menu → "Install ENT Research Data Collection"**
3. Click **Install** — app opens as a standalone window

## Features
- Works **offline** after first load
- Data saved in **localStorage** + synced to **Google Sheets**
- Consent forms uploaded to **Google Drive**
- Installable on any device without an app store
